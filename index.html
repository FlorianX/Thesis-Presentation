HTML5

die Definition

Was ist HTML5?

  • neue Version von HTML
  • ermöglicht bessere Systemintegration
    • Device API, Drag&Drop API, File API
  • (noch) kein Standard
    • W3C und WHATWG
  • Verfasser Ian Hickson
W3C: 2014
WHATWG 2022

Gegenüberstellung

W3C - "HTML5" WHATWG - "HTML"
  • Sub Set der WHATWG-Spezifikation
  • HTML im Mittelpunkt
  • versionsgetrieben
    • Momentaufnahme
  • täglich Korrekturen + Erweiterungen
  • neue JS/DOM API‘s
  • versionslos
    • "Living Standard"
  • Ausrichtung auf Web Applikationen
www.whatwg.org/html
www.w3.org/TR/html5/

  • Microdata

  • Web storage
  • ...

$(element).bind('loadedmetadata', function(){
	$('#duration').html(transSec(element.duration));
});


$('#playButton').click(function(){
	var interval;
	if(element.paused){
		element.play();
		$(this).html('stop!');
		
		interval = setInterval(function(){
			$('#shown').html(transSec(element.currentTime));
		}, 1000);
	}else{
		clearInterval(interval);
		element.pause();
		$(this).html('play!');
	}
});
